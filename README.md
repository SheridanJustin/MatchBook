## What is MatchBook?

Matchbook was an experimental project I wanted to create to test the barriers of lower-level self hosted AI. Given a text query, the app reccomends you 5 books with additonal features such as book favouriting and
"to-read" saving. The app uses Kotlin/Jetpack compose for the UI, Firebase for database read/writing, python for the backend and Ollama for the AI integration. 
Additonally it used Google Books API for the images of each book.To make the project a bit fancier it was hosted on anEC2 instance using CPU only for testing. 
The responses and consistency of the AI heavily depends on the strength of the model. 


---

## Features

**Book Reccomendation**
  - Enter any genre, theme, or description 
  - Get 5 AI-generated real book suggestions

**Book Logging**
  - Save books to a favorites list
  - Add books to a "to-read" list 
  - Mark books as completed
